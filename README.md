# Phoenix Trips

Phoenix Trips is a full-stack travel booking web app built with React, Express, and MongoDB.

## Features

- flight browsing and booking
- hotels and Airbnb stays
- preplanned trip packages
- seat selection flow
- admin and user authentication
- booking management

## Tech Stack

- Frontend: React, Tailwind CSS, Material UI
- Backend: Node.js, Express
- Database: MongoDB

## Local Development

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm start
```

Start the backend:

```bash
npm run server
```

Or use the helper script on Windows:

```powershell
.\scripts\start-all.ps1
```
